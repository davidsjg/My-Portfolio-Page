The New York Times API is utilized to allow the user to search for a given topic and return a requested number of articles.   • Urban El Dorado Application (click to read more)

    Urban El Dorado is an application created by my team in the DU Bootcamp course. It utilizes two APIs (the Census and MapBox) that allow a user to search for a city of interest. The city is shown on an interactive map, which displays and lists nearby schools. Demographics about the area are then pulled and displayed for the user.

  • Work Day Scheduler (click to read more)

    This scheduling calendar uses local storage to maintain an hourly planner. The calendar is dynamic in that it changes color for a given hour based on it's relation to the current time. This allows the user to see what the day still has left, and what has been completed.
